# Karma Devi Trust WordPress Theme

This folder contains a complete ready-to-upload theme package.

## Upload to WordPress
1. Build the ZIP locally by running `./scripts/build-wordpress-theme-zip.sh` (this creates `dist/karma-devi-trust-theme.zip`).
2. In WordPress Admin: Appearance -> Themes -> Add New -> Upload Theme.
3. Activate theme.
4. Create pages with slugs: about, programs, donate, register, contact.
5. Set Settings -> Reading -> Homepage to a static page if desired.
